### Proyecto de investigación
Simulación Evolutiva para Sistemas de Inteligencia Artificial

Proyecto experimental enfocado en la construcción de un entorno evolutivo para agentes inteligentes que interactúan en múltiples nichos simulados.

El sistema explora dinámicas como:

evolución de estrategias

cooperación y competencia

transferencia cultural entre generaciones

mercados de información entre agentes

El modelo incluye:

simulación multi-agente

presión de diversidad evolutiva

penalización de exploit y reward hacking

memoria cultural compartida

mecanismos de meta-gobernanza evolutiva

Inspirado en conceptos como:

Genetic Algorithms

Multi-Agent Systems

Artificial Life

Este proyecto busca explorar cómo emergen comportamientos complejos en poblaciones de agentes adaptativos.
